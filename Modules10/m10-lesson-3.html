Lesson 10.3: java.util (ArrayList နှင့် Scanner)

ကြာမြင့်ချိန်: 25 မိနစ် | အဆင့်: အခြေခံ

၁။ ArrayList ဆိုသည်မှာ ဘာလဲ?

သာမန် Array များသည် အရွယ်အစား အသေ (Fixed size) ဖြစ်သော်လည်း ArrayList သည် လိုအပ်သလို တိုးနိုင်၊ လျှော့နိုင်သော **Dynamic Array** ဖြစ်သည်။ ၎င်းကို အသုံးပြုရန် java.util.ArrayList ကို import လုပ်ရန် လိုအပ်သည်။

ArrayList အသုံးပြုပုံ ဥပမာ


import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // ArrayList တစ်ခု တည်ဆောက်ခြင်း (String သီးသန့်)
        ArrayList fruits = new ArrayList<>();

        // ၁။ add() - အချက်အလက် ထည့်ခြင်း
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Mango");

        // ၂။ get() - Index ဖြင့် ပြန်ယူခြင်း
        System.out.println("ပထမဆုံးသီးနှံ: " + fruits.get(0));

        // ၃။ remove() - ဖျက်ထုတ်ခြင်း
        fruits.remove(1); // Banana ကို ဖျက်မည်

        // ၄။ size() - အရေအတွက် စစ်ခြင်း
        System.out.println("စုစုပေါင်း: " + fruits.size());

        // ၅။ Loop ပတ်၍ အကုန်ထုတ်ခြင်း
        for (String f : fruits) {
            System.out.println(f);
        }
    }
}

၂။ Scanner Class (အသုံးပြုသူဆီမှ Input ယူခြင်း)

Keyboard မှ ရိုက်ထည့်လိုက်သော အချက်အလက်များကို ဖတ်ယူရန် Scanner class ကို သုံးသည်။

Scanner အသုံးပြုပုံ ဥပမာ


import java.util.Scanner;

public class UserInput {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("သင့်နာမည် ရိုက်ထည့်ပါ: ");
        String name = sc.nextLine(); // စာသားဖတ်ခြင်း

        System.out.print("သင့်အသက် ရိုက်ထည့်ပါ: ");
        int age = sc.nextInt(); // ဂဏန်းဖတ်ခြင်း

        System.out.println("မင်္ဂလာပါ " + name + "။ သင်သည် " + age + " နှစ်ရှိပါပြီ။");
        
        sc.close(); // အသုံးပြုပြီးပါက ပိတ်ပေးရန်
    }
}
**Pro Tip:** `ArrayList` မှာ primitive types (int, double, etc.) တွေကို တိုက်ရိုက် သိမ်းလို့မရပါဘူး။ အဲ့ဒီအစား Module 6 မှာ သင်ခဲ့တဲ့ Wrapper Classes (Integer, Double) တွေကို သုံးရပါတယ်။ ဥပမာ- `ArrayList numbers = new ArrayList<>();`
