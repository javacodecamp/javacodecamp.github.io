Lesson 10.1: java.lang (String, Math, Object)

ကြာမြင့်ချိန်: 20 မိနစ် | အဆင့်: အခြေခံ

၁။ java.lang Package ဆိုသည်မှာ ဘာလဲ?

java.lang သည် Java ၏ အခြေခံအကျဆုံး Package ဖြစ်သည်။ ၎င်းတွင် ပါဝင်သော Class များကို အသုံးပြုရန်အတွက် import လုပ်ရန် မလိုပါ။ Java က အလိုအလျောက် ထည့်သွင်းပေးထားပြီး ဖြစ်သည်။

၂။ Object Class (အနွယ်တော်အားလုံး၏ အမိ)

Java တွင် ရှိသမျှ Class အားလုံးသည် Object class ကို တိုက်ရိုက်သော်လည်းကောင်း၊ သွယ်ဝိုက်၍သော်လည်းကောင်း အမွေဆက်ခံထားသည်။

  • toString(): Object တစ်ခုကို String အဖြစ် ပြောင်းလဲရန်။
  • equals(): Object နှစ်ခု တူ/မတူ စစ်ဆေးရန်။

၃။ Math Class (သင်္ချာဆိုင်ရာ တွက်ချက်မှုများ)

Math class တွင် သင်္ချာဆိုင်ရာ လုပ်ဆောင်ချက်များစွာ ပါဝင်သည်။ ၎င်းရှိ method အားလုံးသည် static ဖြစ်သောကြောင့် Object ဆောက်ရန် မလိုဘဲ တိုက်ရိုက်သုံးနိုင်သည်။ 


public class Main {
    public static void main(String[] args) {
        System.out.println("အကြီးဆုံးကိန်း: " + Math.max(10, 20)); // 20
        System.out.println("အပေါင်းကိန်းပြောင်းခြင်း: " + Math.abs(-50)); // 50
        System.out.println("နှစ်ထပ်ကိန်းရင်း: " + Math.sqrt(16)); // 4.0
        System.out.println("ပါဝါတင်ခြင်း: " + Math.pow(2, 3)); // 8.0
        System.out.println("Random ထုတ်ခြင်း: " + Math.random()); // 0.0 to 1.0
    }
}

၄။ System Class

System နှင့် ပတ်သက်သော အချက်အလက်များကို သိရှိရန် သုံးသည်။

  • System.out: Output ထုတ်ရန်။
  • System.in: Input ယူရန်။
  • System.currentTimeMillis(): လက်ရှိအချိန်ကို milliseconds ဖြင့် သိရန်။
**သိမှတ်ဖွယ်:** `String` class သည်လည်း `java.lang` package ထဲတွင် ပါဝင်သောကြောင့် ကျွန်ုပ်တို့သည် String များကို import လုပ်စရာမလိုဘဲ အမြဲတမ်း သုံးနိုင်နေခြင်း ဖြစ်သည်။
