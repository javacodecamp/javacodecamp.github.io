Lesson 10.2: java.time (LocalDate, LocalTime, LocalDateTime)

ကြာမြင့်ချိန်: 20 မိနစ် | အဆင့်: အခြေခံ

၁။ java.time Package ၏ အဓိက Class များ

ရက်စွဲနှင့် အချိန်များကို ကိုင်တွယ်ရန် အောက်ပါ Class များကို အသုံးအများဆုံး ဖြစ်သည်-

  • LocalDate: ရက်စွဲ (နှစ်၊ လ၊ ရက်) ကိုသာ ပြသသည်။
  • LocalTime: အချိန် (နာရီ၊ မိနစ်၊ စက္ကန့်) ကိုသာ ပြသသည်။
  • LocalDateTime: ရက်စွဲနှင့် အချိန် နှစ်ခုလုံးကို ပြသသည်။
  • DateTimeFormatter: ရက်စွဲများကို မိမိစိတ်ကြိုက် ပုံစံ (Format) ပြောင်းလဲရန်။

၂။ လက်ရှိ ရက်စွဲနှင့် အချိန်ကို ရယူခြင်း


import java.time.LocalDate;
import java.time.LocalTime;
import java.time.LocalDateTime;

public class Main {
    public static void main(String[] args) {
        LocalDate myDate = LocalDate.now(); // လက်ရှိ ရက်စွဲ
        LocalTime myTime = LocalTime.now(); // လက်ရှိ အချိန်
        LocalDateTime myFull = LocalDateTime.now(); // နှစ်ခုလုံး

        System.out.println("ရက်စွဲ: " + myDate);
        System.out.println("အချိန်: " + myTime);
        System.out.println("နှစ်ခုလုံး: " + myFull);
    }
}

၃။ DateTimeFormatter ဖြင့် ပုံစံပြောင်းခြင်း

ပုံမှန်အားဖြင့် Java သည် YYYY-MM-DD ပုံစံဖြင့် ပြသသော်လည်း၊ ကျွန်ုပ်တို့သည် မိမိတို့ အလိုရှိရာ ပုံစံသို့ ပြောင်းလဲနိုင်သည်။


import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class Main {
    public static void main(String[] args) {
        LocalDateTime now = LocalDateTime.now();
        
        // ပုံစံသတ်မှတ်ခြင်း (ဥပမာ- ရက်-လ-နှစ် နာရီ:မိနစ်)
        DateTimeFormatter myFormat = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm:ss");
        
        String formattedDate = now.format(myFormat);
        System.out.println("ပုံစံပြောင်းပြီးနောက်: " + formattedDate);
    }
}

၄။ ရက်စွဲများ တွက်ချက်ခြင်း

ရက်ပေါင်းပေါင်းခြင်း သို့မဟုတ် နှုတ်ခြင်းများကို လွယ်ကူစွာ လုပ်ဆောင်နိုင်သည်။


LocalDate today = LocalDate.now();
LocalDate nextWeek = today.plusDays(7); // ၇ ရက် ပေါင်းခြင်း
LocalDate lastMonth = today.minusMonths(1); // ၁ လ နှုတ်ခြင်း
**မှတ်ချက်:** `java.time` package ထဲရှိ Object များသည် **Immutable** ဖြစ်သည်။ ထို့ကြောင့် `plusDays()` ကဲ့သို့ method များကို ခေါ်သည့်အခါ မူလ Object ကို မပြောင်းလဲဘဲ Object အသစ်တစ်ခုကိုသာ ပြန်ပေးသည်။
