Lesson 9.1: try, catch နှင့် finally blocks

ကြာမြင့်ချိန်: 20 မိနစ် | အဆင့်: အခြေခံ

၁။ Exception Handling ဆိုသည်မှာ ဘာလဲ?

Exception ဆိုသည်မှာ Program တစ်ခု Run နေစဉ်အတွင်း ပုံမှန်အလုပ်လုပ်မှုကို အနှောင့်အယှက်ဖြစ်စေသော Error တစ်ခုဖြစ်သည်။ Exception Handling သည် ထို Error များကို စနစ်တကျ ဖမ်းယူဖြေရှင်းပြီး Program ကို ဆက်လက်လည်ပတ်နိုင်အောင် ပြုလုပ်ပေးခြင်း ဖြစ်သည်။

၂။ Try-Catch Block အသုံးပြုပုံ

  • Try block: Error တက်နိုင်ခြေရှိသော Code များကို ဤနေရာတွင် ရေးရသည်။
  • Catch block: Error တက်လာပါက မည်သို့လုပ်ဆောင်မည်ကို ဤနေရာတွင် ရေးရသည်။

Try-Catch ဥပမာ


public class Main {
    public static void main(String[] args) {
        try {
            int result = 10 / 0; // Error တက်မည်
        } catch (ArithmeticException e) {
            System.out.println("Error: သုညဖြင့် စား၍မရပါ။");
        }
        System.out.println("Program ဆက်လက်အလုပ်လုပ်နေသည်။");
    }
}

၃။ Finally Block

finally block အတွင်းရှိ Code များသည် Error တက်သည်ဖြစ်စေ၊ မတက်သည်ဖြစ်စေ **အမြဲတမ်း အလုပ်လုပ်မည်** ဖြစ်သည်။ ပုံမှန်အားဖြင့် ဖိုင်များပိတ်ခြင်း သို့မဟုတ် Connection များကို ပြန်ဖြုတ်ခြင်းအတွက် သုံးသည်။

Finally ဥပမာ


try {
    // some code
} catch (Exception e) {
    // handle error
} finally {
    System.out.println("ဤနေရာသည် အမြဲတမ်း အလုပ်လုပ်သည်။");
}
**မှတ်သားရန်:** `try` တစ်ခုအတွက် `catch` သို့မဟုတ် `finally` တစ်ခုခုသည် မဖြစ်မနေ ပါဝင်ရမည်။ `try` တစ်ခုတည်း သီးသန့် ရေး၍မရပါ။
