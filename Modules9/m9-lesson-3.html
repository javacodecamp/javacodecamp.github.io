Lesson 9.3: အတွေ့ရအများဆုံး Exceptions များနှင့် ၎င်းတို့၏ အကြောင်းအရင်းများ

ကြာမြင့်ချိန်: 10 မိနစ် | အဆင့်: အခြေခံ

၁။ NullPointerException (NPE)

၎င်းသည် Java တွင် အဖြစ်အများဆုံး Error ဖြစ်သည်။ Object တစ်ခုက null ဖြစ်နေစဉ် (Memory ပေါ်တွင် နေရာမရှိစဉ်) ၎င်း၏ method သို့မဟုတ် field ကို ခေါ်ယူအသုံးပြုရန် ကြိုးစားသည့်အခါ ဖြစ်ပေါ်သည်။ 


String s = null;
System.out.println(s.length()); // NullPointerException တက်မည်

၂။ ArithmeticException

သင်္ချာဆိုင်ရာ မှားယွင်းမှုများ (ဥပမာ- ကိန်းပြည့်တစ်ခုကို သုညဖြင့် စားခြင်း) ပြုလုပ်သည့်အခါ ဖြစ်ပေါ်သည်။ 


int result = 10 / 0; // ArithmeticException: / by zero

၃။ ArrayIndexOutOfBoundsException

Array တစ်ခု၏ ရှိရင်းစွဲ အရွယ်အစားထက် ကျော်လွန်သော Index ကို လှမ်းယူသည့်အခါ ဖြစ်ပေါ်သည်။


int[] arr = {1, 2, 3};
System.out.println(arr[5]); // ArrayIndexOutOfBoundsException တက်မည်

၄။ NumberFormatException

မသင့်တော်သော စာသား (String) ကို ဂဏန်း (Number) အဖြစ် ပြောင်းလဲရန် ကြိုးစားသည့်အခါ ဖြစ်ပေါ်သည်။


String s = "ABC";
int i = Integer.parseInt(s); // NumberFormatException တက်မည်

၅။ ClassCastException

Object တစ်ခုကို ၎င်းနှင့် မဆိုင်သော အခြား Type တစ်ခုသို့ အတင်းအကျပ် ပြောင်းလဲ (Casting) သည့်အခါ ဖြစ်ပေါ်သည်။


Object obj = new Integer(10);
String s = (String) obj; // ClassCastException တက်မည်
**Debugging Tip:** Exception တက်လာတဲ့အခါ Console မှာပြတဲ့ **Stack Trace** ကို သေချာဖတ်ပါ။ ဘယ် Exception အမျိုးအစားလဲနဲ့ ဘယ် Line နံပါတ်မှာ Error တက်နေလဲဆိုတာကို အတိအကျ ပြောပြပေးပါတယ်။
