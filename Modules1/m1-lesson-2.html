Lesson 1.2: Data Types (အချက်အလက် အမျိုးအစားများ) နှင့် Variables

ကြာမြင့်ချိန်: 25 မိနစ် | အဆင့်: စတင်သူ

၁။ Variables ဆိုသည်မှာ ဘာလဲ?

Variable ဆိုသည်မှာ Program အတွင်း အချက်အလက် (Data) များကို ခေတ္တသိမ်းဆည်းထားသည့် Memory နေရာတစ်ခုကို ပေးထားသော အမည် (Name) ဖြစ်သည်။ Variable ကို အသုံးမပြုမီ ကြေညာ (Declare) ရမည်။

Variable ကြေညာပုံ


// ဖွဲ့စည်းပုံ: [Data Type] [Variable Name] = [Value];
int age = 30;
double price = 19.99;
char initial = 'K';

၂။ Java ၏ Data Types အမျိုးအစားများ

Java တွင် Data Types နှစ်မျိုးရှိသည်။

A. Primitive Data Types (မူလအမျိုးအစားများ)

၎င်းတို့သည် Memory ထဲတွင် Value ကို တိုက်ရိုက်သိမ်းဆည်းပြီး၊ အရွယ်အစား (Size) ကို ပုံသေ သတ်မှတ်ထားသည်။ Java တွင် Primitive Type ၈ မျိုးရှိသည်။

Type ဖော်ပြချက် အရွယ်အစား (Size)
byte ဂဏန်းအငယ်စားများ (Small integers) 1 byte
short အလတ်စား ဂဏန်းများ 2 bytes
int အများဆုံးသုံးသော ဂဏန်းအပြည့် (Integers) 4 bytes
long ဂဏန်းအကြီးစားများ 8 bytes
float ဒသမကိန်း (Fractional numbers) 4 bytes
double တိကျမှုမြင့်သော ဒသမကိန်း (Higher precision) 8 bytes
char တစ်လုံးတည်းသော စာလုံး (Single character) 2 bytes (Unicode)
boolean မှန်/မှား တန်ဖိုး (True/False) 1 bit

B. Reference Data Types (ရည်ညွှန်း အမျိုးအစားများ)

၎င်းတို့သည် တန်ဖိုး (Value) ကို တိုက်ရိုက်မသိမ်းဆည်းဘဲ၊ Memory ထဲရှိ Object တည်ရှိရာ နေရာ (Address/Reference) ကို သိမ်းဆည်းထားသည်။

  • **ဥပမာများ:** Classes, Interfaces, Arrays, နှင့် **String** တို့ဖြစ်သည်။
  • **String** သည် စာလုံးများစွာကို သိမ်းဆည်းရန် အသုံးအများဆုံး Reference Type ဖြစ်သည်။

Reference Type ဥပမာ


// String သည် Class ဖြစ်၍ Reference Type ဖြစ်သည်
String name = "Java Code"; 

// String ကို အသုံးပြု၍ အချက်အလက် ထုတ်ပြခြင်း
System.out.println(name);
Primitive Type များကို စတင်ကြေညာသောအခါ တန်ဖိုးမပေးထားလျှင် (ဥပမာ- int count;) Default Value (သုည သို့မဟုတ် မှား) ရရှိသည်။ Local Variables များတွင်မူ တန်ဖိုးမပေးဘဲ သုံးလျှင် Error ဖြစ်သည်။
