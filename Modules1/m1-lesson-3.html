Lesson 1.3: Type Casting (အမျိုးအစား ပြောင်းလဲခြင်း) နှင့် Operators

ကြာမြင့်ချိန်: 30 မိနစ် | အဆင့်: စတင်သူ

၁။ Type Casting (Data အမျိုးအစား ပြောင်းလဲခြင်း)

Type Casting ဆိုသည်မှာ Data Type တစ်ခုကို နောက်ထပ် Data Type တစ်ခုသို့ ပြောင်းလဲခြင်းဖြစ်သည်။ ၎င်းတွင် အဓိကအားဖြင့် အမျိုးအစား နှစ်မျိုးရှိသည်။

A. Widening/Implicit Casting (အလိုအလျောက် ပြောင်းလဲခြင်း)

အရွယ်အစားသေးငယ်သော Type မှ အရွယ်အစား ပိုကြီးသော Type သို့ ပြောင်းလဲခြင်းဖြစ်သည်။ (ဥပမာ: int မှ double) ဤသည်မှာ Data ပျောက်ဆုံးမှု မရှိသောကြောင့် Java က အလိုအလျောက် ခွင့်ပြုသည်။

Widening Casting ဥပမာ


int myInt = 9;
double myDouble = myInt; // Automatic casting
System.out.println(myInt); 
System.out.println(myDouble);

Output ရလဒ်

9
9.0

B. Narrowing/Explicit Casting (အတင်းအဓမ္မ ပြောင်းလဲခြင်း)

အရွယ်အစားကြီးသော Type မှ သေးငယ်သော Type သို့ ပြောင်းလဲခြင်းဖြစ်သည်။ (ဥပမာ: double မှ int) ဤသည်မှာ Data ပျောက်ဆုံးနိုင်သောကြောင့် (ဒသမကိန်းများ ပျောက်သွားနိုင်သည်)၊ **ကွင်းစကွင်းပိတ်** ထဲတွင် Type အသစ်ကို ထည့်၍ Manual လုပ်ပေးရသည်။

Narrowing Casting ဥပမာ


double myDouble = 9.99;
int myInt = (int) myDouble; // Manual casting လိုအပ်သည်။
System.out.println(myDouble);
System.out.println(myInt); // ဒသမကိန်း ပျောက်သွားသည်။

Output ရလဒ်

9.99
9

၂။ Operators များ (တွက်ချက်မှု သင်္ကေတများ)

Operator ဆိုသည်မှာ Variable များ သို့မဟုတ် တန်ဖိုးများပေါ်တွင် တွက်ချက်မှု (Operation) များ ပြုလုပ်ရန် အသုံးပြုသော အထူးသင်္ကေတများ ဖြစ်သည်။

A. Arithmetic Operators (ဂဏန်းတွက်ချက်မှု)

သာမန် ဂဏန်းပေါင်း၊ နှုတ်၊ မြှောက်၊ စားခြင်းတို့အတွက် အသုံးပြုသည်။

Code


int a = 10;
int b = 3;
System.out.println("ပေါင်းခြင်း: " + (a + b)); // 13
System.out.println("စားကြွင်း: " + (a % b)); // 10 ကို 3 ဖြင့် စားလျှင် ကြွင်းသော 1

Output ရလဒ်

ပေါင်းခြင်း: 13
စားကြွင်း: 1

B. Comparison/Relational Operators (နှိုင်းယှဉ်မှု)

တန်ဖိုးနှစ်ခုကို နှိုင်းယှဉ်ပြီး ရလဒ်ကို boolean (True သို့မဟုတ် False) အဖြစ် ပြန်ပေးသည်။

Code


int x = 5;
int y = 8;
System.out.println("တူညီပါသလား (==): " + (x == y)); // False
System.out.println("မတူညီပါဘူးလား (!=): " + (x != y)); // True
System.out.println("သေးပါသလား (<): " + (x < y)); // True

Output ရလဒ်

တူညီပါသလား (==): false
မတူညီပါဘူးလား (!=): true
သေးပါသလား (<): true

C. Logical Operators (ယုတ္တိဗေဒ)

Boolean တန်ဖိုးများ (True/False) နှစ်ခု သို့မဟုတ် နှစ်ခုထက်ပိုသော အခြေအနေများကို ပေါင်းစပ်ရာတွင် အသုံးပြုသည်။

Code


boolean isJavaFun = true;
boolean isDifficult = false;

// && (AND): နှစ်ခုစလုံး True ဖြစ်မှ True ဖြစ်မည်။
System.out.println("AND (&&): " + (isJavaFun && isDifficult)); // False

// || (OR): တစ်ခုခု True ဖြစ်ရင်ကို True ဖြစ်မည်။
System.out.println("OR (||): " + (isJavaFun || isDifficult)); // True

// ! (NOT): တန်ဖိုးကို ပြောင်းပြန်လှန်သည်။
System.out.println("NOT (!): " + (!isDifficult)); // True

Output ရလဒ်

AND (&&): false
OR (||): true
NOT (!): true
Ternary Operator (? :) သည် if-else statement ကို တိုတိုလေး ရေးသားနိုင်စေသည်။ (ဥပမာ: String result = (score > 50) ? "Pass" : "Fail";)
