Lesson 1.1: Java ၏ Program ဖွဲ့စည်းပုံနှင့် Setup

ကြာမြင့်ချိန်: 15 မိနစ် | အဆင့်: စတင်သူ

၁။ Java Development Environment Setup

Java Code များကို ရေးသား၊ Compile (ဘာသာပြန်) နှင့် Run (လည်ပတ်) နိုင်ရန်အတွက် အဓိကအားဖြင့် အောက်ပါ Tools များ လိုအပ်ပါသည်။

  • **JDK (Java Development Kit):** Java Code များကို Compile လုပ်ရန်နှင့် Run ရန် လိုအပ်သော Tool များ၊ Libraries များနှင့် JVM (Java Virtual Machine) တို့ ပါဝင်သည်။
  • **IDE (Integrated Development Environment):** Code များကို ပိုမိုလွယ်ကူလျင်မြန်စွာ ရေးသားနိုင်ရန် ကူညီပေးသော Software များ (ဥပမာ: IntelliJ IDEA, Eclipse, VS Code)။

၂။ Java Program ၏ အခြေခံဖွဲ့စည်းပုံ

ရိုးရှင်းသော Java Program တစ်ခုသည် အောက်ပါ အစိတ်အပိုင်းများဖြင့် ဖွဲ့စည်းထားပါသည်။

A. Class (အတန်း)

Java ရှိ Code အားလုံးသည် Class တစ်ခု၏ အတွင်း၌ တည်ရှိရမည်။ Class သည် Program အတွက် Blueprint သို့မဟုတ် အိမ်ခေါင်းစဉ်ဖြစ်သည်။

B. main() Method

Program စတင် လည်ပတ်သည့် နေရာ (Starting Point) ဖြစ်သည်။ Java Virtual Machine (JVM) သည် Program ကို စတင်သည့်အခါ ဤနေရာမှ ရှာဖွေစတင်သည်။ ၎င်းကို ပုံသေအားဖြင့် public static void main(String[] args) ဟု ရေးသားရသည်။

Java Code ဥပမာ: "Hello World"


// 1. Class ၏ အမည်ကို ပေးထားသည်။ (ဖိုင်အမည်နှင့် တူရမည် - Hello.java)
public class Hello { 

    // 2. Program စတင်ရာ main method
    public static void main(String[] args) {
        
        // 3. Statement တစ်ခုဖြစ်ပြီး Console သို့ စာသားထုတ်ပြသည်။
        System.out.println("Hello, Java World!"); 
    }
}

Output ရလဒ်

Hello, Java World!

၃။ Compilation & Execution Process

  1. **ရေးသားခြင်း:** Developer က Source Code (`Hello.java`) ကို ရေးသားသည်။
  2. **Compilation:** JDK အတွင်းရှိ Compiler (javac) ကို အသုံးပြု၍ Source Code ကို **Bytecode** (`Hello.class`) အဖြစ် ပြောင်းလဲသည်။
  3. **Execution:** Bytecode ကို JVM (Java Virtual Machine) မှတစ်ဆင့် ကွန်ပျူတာပေါ်တွင် လည်ပတ်စေသည်။ ဤအဆင့်ကြောင့် Java ကို Platform-independent ဖြစ်စေသည်။
Java သည် "Write Once, Run Anywhere" (WORA) သဘောတရားကို ကျင့်သုံးပြီး၊ Bytecode ကြောင့် မည်သည့် OS (Windows, Mac, Linux) တွင်မဆို Run နိုင်သည်။
