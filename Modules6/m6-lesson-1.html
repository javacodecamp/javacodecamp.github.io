Lesson 6.1: Wrapper Classes (Integer, Double, Character)

ကြာမြင့်ချိန်: 15 မိနစ် | အဆင့်: အခြေခံ

၁။ Wrapper Class ဆိုသည်မှာ ဘာလဲ?

Java တွင် Primitive Data Types (int, double, char, etc.) များသည် Object များ မဟုတ်ကြပါ။ သို့သော် တစ်ခါတစ်ရံတွင် ၎င်းတို့ကို Object ကဲ့သို့ အသုံးပြုရန် လိုအပ်တတ်သည် (ဥပမာ- Collections များတွင် သုံးသည့်အခါ)။ ထိုသို့ အသုံးပြုနိုင်ရန် Primitive တန်ဖိုးများကို Object အဖြစ် "ထုပ်ပိုး" ပေးထားသော Class များကို Wrapper Classes ဟု ခေါ်သည်။

၂။ အသုံးများသော Wrapper Classes များ

Primitive Type Wrapper Class
byte Byte
short Short
int Integer
long Long
float Float
double Double
char Character
boolean Boolean

၃။ Wrapper Class ၏ အသုံးဝင်ပုံ

Wrapper classes များသည် သာမန် Primitive types များထက် ပိုမိုစွမ်းဆောင်နိုင်သော Method များစွာ ပါဝင်သည်။ အသုံးအများဆုံးမှာ Parsing (စာသားကို ဂဏန်းပြောင်းခြင်း) ဖြစ်သည်။

Wrapper Class Method များ အသုံးပြုပုံ


public class Main {
    public static void main(String[] args) {
        // ၁။ String ကို int ပြောင်းခြင်း
        String s = "123";
        int num = Integer.parseInt(s);
        
        // ၂။ Integer Object ဖန်တီးခြင်း
        Integer myInt = 50; 
        
        // ၃။ Method များ အသုံးပြုခြင်း
        System.out.println("ဂဏန်း: " + num);
        System.out.println("Object အဖြစ်ပြောင်းလဲမှု: " + myInt.doubleValue());
        System.out.println("စာလုံးဖြစ်သလား: " + Character.isLetter('A'));
    }
}

Output ရလဒ်

ဂဏန်း: 123
Object အဖြစ်ပြောင်းလဲမှု: 50.0
စာလုံးဖြစ်သလား: true
**မှတ်ချက်:** Wrapper class အမည်များသည် Primitive type များနှင့် အတူတူပင် ဖြစ်သော်လည်း ရှေ့စာလုံးမှာ **အကြီး (Uppercase)** ဖြစ်ရမည်ကို သတိပြုပါ။ (ဥပမာ- int သည် Integer၊ char သည် Character ဖြစ်သည်)။
