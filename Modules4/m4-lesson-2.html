Lesson 4.2: Method Overloading, Varargs နှင့် Pass-by-value

ကြာမြင့်ချိန်: 27 မိနစ် | အဆင့်: အလယ်အလတ်

၁။ Method Overloading ဆိုသည်မှာ ဘာလဲ?

Class တစ်ခုတည်းအတွင်း၌ **Method Name အတူတူ** ကို အသုံးပြုသော်လည်း ၎င်းတို့လက်ခံသော **Parameters (Arguments)** များ မတူညီအောင် ဖန်တီးခြင်းကို ခေါ်သည်။ ၎င်းသည် ပရိုဂရမ်ကို ပိုမိုဖတ်ရလွယ်ကူစေသည်။

Method Overloading ဥပမာ


public class MathUtils {
    // ကိန်းပြည့်နှစ်ခုပေါင်းသော method
    static int add(int a, int b) {
        return a + b;
    }

    // ဒသမကိန်းနှစ်ခုပေါင်းသော method (နာမည်တူသော်လည်း parameter မတူပါ)
    static double add(double a, double b) {
        return a + b;
    }

    public static void main(String[] args) {
        System.out.println("Integer Sum: " + add(5, 10));
        System.out.println("Double Sum: " + add(5.5, 4.5));
    }
}

Output ရလဒ်

Integer Sum: 15
Double Sum: 10.0

၂။ Variable Arguments (Varargs)

Method တစ်ခုကို ခေါ်ယူသည့်အခါ Parameter အရေအတွက် မည်မျှလာမည်ကို အတိအကျမသိနိုင်ပါက ... (ellipsis) ကို အသုံးပြုနိုင်သည်။

Varargs ဥပမာ


static void printNumbers(int... nums) {
    for (int n : nums) {
        System.out.print(n + " ");
    }
    System.out.println();
}

public static void main(String[] args) {
    printNumbers(1, 2);
    printNumbers(10, 20, 30, 40);
}

Output ရလဒ်

1 2 
10 20 30 40

၃။ Pass-by-value သဘောတရား

Java တွင် Method ဆီသို့ Parameter ပေးပို့သည့်အခါ **Value (တန်ဖိုး)** ကိုသာ ကူးယူပေးပို့ခြင်းဖြစ်သည်။ မူရင်း Variable ထဲရှိ တန်ဖိုးသည် Method အတွင်း ပြောင်းလဲမှုကြောင့် လိုက်ပါပြောင်းလဲခြင်း မရှိပါ။

Pass-by-value ဥပမာ


static void changeValue(int x) {
    x = 100; // x ကို ၁၀၀ ပြောင်းလိုက်သည်
}

public static void main(String[] args) {
    int num = 50;
    changeValue(num); // num ရဲ့ copy (၅၀) ကိုပဲ ပေးလိုက်တာပါ
    System.out.println("Original Num: " + num);
}

Output ရလဒ်

Original Num: 50
**မှတ်ချက်:** Method Overloading လုပ်သည့်အခါ Return Type မတူရုံမျှဖြင့် မရပါ။ Parameter list (အရေအတွက် သို့မဟုတ် အမျိုးအစား) သိသိသာသာ ကွဲပြားရန် လိုအပ်ပါသည်။
