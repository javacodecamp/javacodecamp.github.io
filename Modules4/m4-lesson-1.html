Lesson 4.1: Method များ ကြေညာခြင်းနှင့် အသုံးပြုခြင်း

ကြာမြင့်ချိန်: 20 မိနစ် | အဆင့်: အခြေခံ

၁။ Method ဆိုသည်မှာ ဘာလဲ?

Method ဆိုသည်မှာ အလုပ်တစ်ခုကို လုပ်ဆောင်ရန် စုစည်းထားသော Code အစုအဝေး ဖြစ်သည်။ ၎င်းကို Function ဟုလည်း ခေါ်ကြသည်။ Method များကို အသုံးပြုခြင်းဖြင့် Code များကို ထပ်ခါတလဲလဲ ရေးစရာမလိုဘဲ လိုအပ်သည့်နေရာမှ လှမ်းခေါ် (Call) နိုင်သည်။

၂။ Method ဖွဲ့စည်းပုံ (Syntax)

Method တစ်ခုတွင် အဓိက အစိတ်အပိုင်း ၄ ခု ပါဝင်သည်။

  • **Return Type:** Method က အလုပ်လုပ်ပြီးလျှင် ပြန်ပေးမည့် Data Type (ဥပမာ: int, String)။ ဘာမှပြန်မပေးလိုလျှင် void ဟု ရေးရသည်။
  • **Method Name:** Method ကို ခေါ်ယူရန် ပေးထားသော အမည်။
  • **Parameters:** Method ဆီသို့ ပေးပို့လိုက်သော အချက်အလက်များ (မရှိလျှင် ကွင်းလွတ် ထားနိုင်သည်)။
  • **Method Body:** အလုပ်လုပ်မည့် Code များ။

ဥပမာ ၁- ရိုးရှင်းသော Method (void type)


public class Main {
    // ၁။ Method ကို ကြေညာခြင်း
    static void sayHello(String name) {
        System.out.println("Hello " + name + "!");
    }

    public static void main(String[] args) {
        // ၂။ Method ကို လှမ်းခေါ်ခြင်း (Calling)
        sayHello("Mg Mg");
        sayHello("Su Su");
    }
}

Output ရလဒ်

Hello Mg Mg!
Hello Su Su!

၃။ Return Type ပါဝင်သော Method

အကယ်၍ Method တစ်ခုဆီမှ အဖြေတစ်ခုကို ပြန်လည်ရယူလိုပါက return keyword ကို အသုံးပြုရသည်။

ဥပမာ ၂- ကိန်းနှစ်ခုပေါင်းပြီး အဖြေပြန်ပေးသော Method


public class Calculation {
    // int တန်ဖိုး ပြန်ပေးမည့် method
    static int addNumbers(int x, int y) {
        return x + y; // အဖြေကို ပြန်ပေးလိုက်သည်
    }

    public static void main(String[] args) {
        int result = addNumbers(10, 20); // အဖြေကို variable ထဲ သိမ်းသည်
        System.out.println("ပေါင်းလဒ်: " + result);
    }
}

Output ရလဒ်

ပေါင်းလဒ်: 30
**မှတ်ချက်:** static ဟု ရေးရခြင်းမှာ main method ထဲမှနေ၍ Object မဆောက်ဘဲ တိုက်ရိုက်ခေါ်သုံးနိုင်ရန် ဖြစ်သည်။ Object-Oriented အပိုင်းတွင် ၎င်းအကြောင်းကို အသေးစိတ် ထပ်မံလေ့လာရပါမည်။
