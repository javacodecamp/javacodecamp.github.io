Lesson 8.1: Package Declaration, Importing Classes & FQNs

ကြာမြင့်ချိန်: 15 မိနစ် | အဆင့်: အခြေခံ

၁။ Package ဆိုသည်မှာ ဘာလဲ?

Package ဆိုသည်မှာ တစ်ခုနှင့်တစ်ခု ဆက်စပ်နေသော Class များ၊ Interfaces များနှင့် Sub-packages များကို စုစည်းထားသည့် **Folder** တစ်ခုဖြစ်သည်။ ၎င်းသည် နာမည်တူ Class များကြား ပဋိပက္ခမဖြစ်အောင် (Naming Conflicts) ကာကွယ်ပေးသည်။

၂။ Package ကြေညာခြင်း (Package Declaration)

Java ဖိုင်တစ်ခုကို Package တစ်ခုအတွင်း ထည့်လိုလျှင် Code ၏ အပေါ်ဆုံးတွင် package keyword ကို အသုံးပြုရသည်။


package com.myproject.utils; // Package ကြေညာခြင်း

public class Calculator {
    // ... code ...
}

၃။ Importing Classes

အခြား Package ထဲတွင် ရှိသော Class တစ်ခုကို အသုံးပြုလိုလျှင် import keyword ဖြင့် ခေါ်ယူရသည်။

  • Class တစ်ခုတည်းကို Import လုပ်ခြင်း: import java.util.Scanner;
  • Package တစ်ခုလုံးရှိ Class အားလုံးကို Import လုပ်ခြင်း: import java.util.*;

၄။ Fully Qualified Name (FQN)

import မသုံးဘဲ Class တစ်ခု၏ Package လိပ်စာအပြည့်အစုံကို ရေးသားပြီး ခေါ်ယူခြင်းကို Fully Qualified Name ဟု ခေါ်သည်။ 


public class Main {
    public static void main(String[] args) {
        // Import မသုံးဘဲ FQN ဖြင့် Scanner ကို ခေါ်သုံးခြင်း
        java.util.Scanner sc = new java.util.Scanner(System.in);
    }
}
**မှတ်သားရန်:** java.lang package ထဲတွင်ရှိသော Class များ (ဥပမာ- String, System) ကိုမူ Java က အလိုအလျောက် Import လုပ်ပေးထားသဖြင့် ထပ်မံ Import လုပ်ရန် မလိုပါ။
