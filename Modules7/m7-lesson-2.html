Lesson 7.2: StringBuilder (append,insert,reverse)

ကြာမြင့်ချိန်: 15 မိနစ် | အဆင့်: အခြေခံ

၁။ StringBuilder ဆိုသည်မှာ ဘာလဲ?

StringBuilder ဆိုသည်မှာ **Mutable** (ပြင်ဆင်၍ရသော) စာသား Object တစ်ခုဖြစ်သည်။ ၎င်းကို အသုံးပြုခြင်းဖြင့် စာသားအသစ်များ ထပ်ပေါင်းခြင်း၊ ဖြတ်ထုတ်ခြင်းတို့ကို Memory အပိုမကုန်ဘဲ လုပ်ဆောင်နိုင်သည်။

၂။ အသုံးများသော StringBuilder Methods များ

လက်တွေ့အသုံးပြုပုံ ဥပမာ


public class Main {
    public static void main(String[] args) {
        // StringBuilder Object တစ်ခု တည်ဆောက်ခြင်း
        StringBuilder sb = new StringBuilder("Hello");

        // ၁။ append() - စာသား နောက်ကနေ ပေါင်းထည့်ခြင်း
        sb.append(" Java"); 
        System.out.println("Append ပြီးနောက်: " + sb); // Hello Java

        // ၂။ insert() - ကြားထဲသို့ စာသား ထိုးထည့်ခြင်း
        sb.insert(5, " World");
        System.out.println("Insert ပြီးနောက်: " + sb); // Hello World Java

        // ၃။ reverse() - စာသားကို နောက်ပြန်လှည့်ခြင်း
        sb.reverse();
        System.out.println("Reverse ပြီးနောက်: " + sb); // avaJ dlroW olleH

        // ၄။ toString() - StringBuilder မှ ပုံမှန် String သို့ ပြန်ပြောင်းခြင်း
        String finalResult = sb.toString();
    }
}

Output ရလဒ်

Append ပြီးနောက်: Hello Java
Insert ပြီးနောက်: Hello World Java
Reverse ပြီးနောက်: avaJ dlroW olleH

၃။ String နှင့် StringBuilder မည်သည်ကို သုံးသင့်သနည်း?

  • String: စာသားကို တစ်ခါပဲ သတ်မှတ်ပြီး နောက်ထပ် မပြင်တော့ဘူးဆိုရင် သုံးပါ။
  • StringBuilder: Loop တစ်ခုထဲမှာ စာသားတွေကို ခဏခဏ ပေါင်းနေရမယ်ဆိုရင် (ဥပမာ- စာအုပ်တစ်အုပ်စာ စာသားတွေကို ပေါင်းခြင်း) StringBuilder ကို မဖြစ်မနေ သုံးသင့်ပါတယ်။
**သိထားရန်:** Multi-threading (အလုပ်တွေအများကြီး တစ်ပြိုင်တည်းလုပ်တဲ့ ပရိုဂရမ်) တွေမှာဆိုရင်တော့ `StringBuilder` အစား Thread-safe ဖြစ်တဲ့ `StringBuffer` ကို သုံးလေ့ရှိပါတယ်။
