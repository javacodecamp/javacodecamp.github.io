Lesson 7.1: String Immutability နှင့် String Methods

ကြာမြင့်ချိန်: 25 မိနစ် | အဆင့်: အခြေခံ

၁။ Java String ဆိုသည်မှာ ဘာလဲ?

Java တွင် String ဆိုသည်မှာ စာလုံး (Characters) များ စုစည်းထားသော Object တစ်ခု ဖြစ်သည်။ ၎င်းကို " " (Double Quotes) ဖြင့် ရေးသားရသည်။

၂။ String Immutability (ပြောင်းလဲ၍မရသော သဘာဝ)

Java တွင် String Object တစ်ခုကို ဖန်တီးပြီးပါက ၎င်း၏ တန်ဖိုးကို **ထပ်မံ ပြောင်းလဲ၍ မရပါ (Immutable)**။ စာသားတစ်ခုကို ပြင်လိုက်တိုင်း Java က ရှိပြီးသား String ကို မပြင်ဘဲ Memory ပေါ်တွင် String အသစ်တစ်ခုကိုသာ ထပ်မံ ဖန်တီးပေးခြင်း ဖြစ်သည်။

Immutability ဥပမာ


String s = "Java";
s.concat(" Code"); // s ကို Code နှင့် ပေါင်းသည်
System.out.println(s); // Output က Java ပဲ ထွက်ပါလိမ့်မည်

*မှတ်ချက်: "Java Code" ဆိုတဲ့ အဖြေကို လိုချင်ရင် variable အသစ်ထဲမှာ ပြန်သိမ်းရပါမယ်။*

၃။ အသုံးများသော String Methods များ

String Object များတွင် စာသားများကို ကိုင်တွယ်ရန် အလွန်အသုံးဝင်သော Methods များ ပါဝင်သည်။

လက်တွေ့အသုံးပြုပုံ


String text = "Java Programming";

System.out.println("အရှည်: " + text.length()); // 16
System.out.println("အကြီးပြောင်းရန်: " + text.toUpperCase()); // JAVA PROGRAMMING
System.out.println("Index 0 မှ စာလုံး: " + text.charAt(0)); // J
System.out.println("စာသားဖြတ်ထုတ်ခြင်း: " + text.substring(0, 4)); // Java
System.out.println("တူမတူ စစ်ဆေးခြင်း: " + text.equals("java")); // false (Case sensitive)
**သတိထားရန်:** String နှစ်ခုကို တူ/မတူ စစ်ဆေးသည့်အခါ == ကို မသုံးဘဲ .equals() ကိုသာ အမြဲသုံးသင့်သည်။ == သည် Memory Address တူ/မတူကိုသာ စစ်ဆေးသောကြောင့် ဖြစ်သည်။
