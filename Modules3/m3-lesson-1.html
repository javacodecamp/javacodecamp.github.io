Lesson 3.1: One-D & Multi-D Arrays (အစုအဝေးများ)

ကြာမြင့်ချိန်: 20 မိနစ် | အဆင့်: အခြေခံ

၁။ Array ဆိုသည်မှာ ဘာလဲ?

Array ဆိုသည်မှာ **တူညီသော Data Type** ရှိသည့် တန်ဖိုးများကို အစုအဝေးတစ်ခုတည်းအဖြစ် သိမ်းဆည်းထားသော အရာဖြစ်သည်။ Array တစ်ခုကို တည်ဆောက်ပြီးပါက ၎င်း၏ **Size (အရွယ်အစား)** ကို ထပ်မံတိုးချဲ့ခြင်း သို့မဟုတ် လျှော့ချခြင်း ပြုလုပ်၍မရပါ။

၂။ One-dimensional Array (တစ်ဖက်သတ် Array)

အသုံးအများဆုံး Array ပုံစံဖြစ်ပြီး စာရင်းတစ်ခုကဲ့သို့ အချက်အလက်များကို သိမ်းဆည်းသည်။

Array ကြေညာခြင်းနှင့် အသုံးပြုခြင်း


// ၁။ Array ကြေညာခြင်း (အရွယ်အစား ၃ ခုစာ)
int[] numbers = new int[3];

// ၂။ တန်ဖိုးများ ထည့်သွင်းခြင်း (Index သည် 0 မှ စသည်)
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;

// ၃။ တန်ဖိုးကို ပြန်ထုတ်ကြည့်ခြင်း
System.out.println("ပထမကိန်း: " + numbers[0]);
System.out.println("Array အရှည်: " + numbers.length);

Output ရလဒ်

ပထမကိန်း: 10
Array အရှည်: 3

၃။ Multi-dimensional Array (ဘက်စုံ Array)

Array တစ်ခု၏ အတွင်း၌ နောက်ထပ် Array များ ထပ်မံပါဝင်ခြင်းဖြစ်သည်။ အသုံးအများဆုံးမှာ ဇယား (Table) ပုံစံကဲ့သို့ Row နှင့် Column ပါဝင်သော **2D Array** ဖြစ်သည်။

[0], [0][1]]

2D Array ဥပမာ


// Row ၂ ခု နှင့် Column ၃ ခု ပါသော 2D Array
int[][] matrix = {
    {1, 2, 3}, // Row 0
    {4, 5, 6}  // Row 1
};

// Row 1, Column 2 မှ တန်ဖိုး (6) ကို ထုတ်ကြည့်ခြင်း
System.out.println("တန်ဖိုး: " + matrix[1][2]);

Output ရလဒ်

တန်ဖိုး: 6
Array ၏ Index သည် အမြဲတမ်း **0** မှ စတင်ကြောင်း မမေ့ပါနှင့်။ အကယ်၍ Array အရှည် ၅ ခုရှိလျှင် နောက်ဆုံး Index သည် **4** သာ ဖြစ်သည်။
