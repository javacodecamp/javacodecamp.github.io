Lesson 3.2: Array အမှားများ နှင့် For-each Loop

ကြာမြင့်ချိန်: 18 မိနစ် | အဆင့်: အခြေခံ

၁။ ArrayIndexOutOfBoundsException ဆိုသည်မှာ ဘာလဲ?

ဒါဟာ Java Beginner တွေ အဖြစ်အများဆုံး Error တစ်ခုပါ။ Array ထဲမှာ မရှိတဲ့ **Index** (နေရာ) ကို လှမ်းခေါ်တဲ့အခါမှာ ဒီ Exception တက်လာပါတယ်။

Error ဖြစ်စေမည့် ဥပမာ


int[] myNumbers = {10, 20, 30}; // အရှည် ၃ ခု (Index 0, 1, 2 ပဲရှိသည်)

// Index 3 ကို ခေါ်ကြည့်ခြင်း (မရှိသောနေရာ)
System.out.println(myNumbers[3]);

ရလဒ် (Console Error)

Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: Index 3 out of bounds for length 3

၂။ Array ၏ Default Values များ

Array တစ်ခုကို တည်ဆောက်ပြီး တန်ဖိုးချက်ချင်း မထည့်သေးဘူးဆိုရင် Java က အောက်ပါအတိုင်း Default တန်ဖိုးတွေ သတ်မှတ်ပေးထားပါတယ်။

  • ဂဏန်းများ (int, double, etc.) : 0 သို့မဟုတ် 0.0
  • Boolean : false
  • Reference (String, Objects) : null

၃။ Enhanced for-loop (For-each Loop)

Array ထဲက အချက်အလက်အားလုံးကို တစ်ခုချင်းစီ ထုတ်ကြည့်ချင်တဲ့အခါ ရိုးရိုး for loop ထက် ပိုပြီး ရေးရတာ ရိုးရှင်းတဲ့ **For-each** ကို သုံးနိုင်ပါတယ်။

For-each Loop ဥပမာ


String[] fruits = {"Apple", "Banana", "Orange"};

// "fruits Array ထဲက တစ်ခုချင်းစီကို f ထဲထည့်ပြီး ပတ်မယ်" လို့ အဓိပ္ပာယ်ရသည်
for (String f : fruits) {
    System.out.println("သစ်သီး: " + f);
}

Output ရလဒ်

သစ်သီး: Apple
သစ်သီး: Banana
သစ်သီး: Orange
**မှတ်ချက်:** For-each loop ကို Array ထဲက တန်ဖိုးတွေကို ဖတ်ဖို့ (Read-only) အတွက်သာ သုံးသင့်ပါတယ်။ Array ထဲက တန်ဖိုးတွေကို ပြောင်းလဲချင်ရင်တော့ Index ပါတဲ့ ရိုးရိုး For loop ကိုပဲ သုံးရပါမယ်။
