Lesson 2.3: For, While, Do-While Loops (ထပ်ခါတလဲလဲ လုပ်ဆောင်ခြင်း)

ကြာမြင့်ချိန်: 35 မိနစ် | အဆင့်: အခြေခံ

၁။ Loop ဆိုသည်မှာ ဘာလဲ?

Loop ဆိုသည်မှာ သတ်မှတ်ထားသော Code Block တစ်ခုကို **Condition (အခြေအနေ)** မမှန်တော့သည်အထိ သို့မဟုတ် သတ်မှတ်ထားသော အကြိမ်ရေ ပြည့်သည်အထိ **ထပ်ခါတလဲလဲ** လုပ်ဆောင်စေရန် အသုံးပြုသော Control Structure ဖြစ်သည်။

၂။ The `for` Loop

အကြိမ်ရေ **မည်မျှ လည်ပတ်ရမည်** ကို ကြိုတင်သိရှိထားသည့်အခါ for loop ကို အများဆုံး အသုံးပြုသည်။ ၎င်းတွင် အစိတ်အပိုင်း ၃ ခု ပါဝင်သည်။

  1. **Initialization (စတင်ခြင်း):** Loop စတင်ရာ နေရာ (ဥပမာ: int i = 0;)
  2. **Condition (အခြေအနေ):** Loop ဆက်လက် လည်ပတ်ရမည့် အခြေအနေ (ဥပမာ: i < 5;)
  3. **Iteration (တိုးခြင်း/လျော့ခြင်း):** Loop တစ်ကြိမ်ပြီးတိုင်း လုပ်ဆောင်မည့် အပြောင်းအလဲ (ဥပမာ: i++)

`for` Loop ဥပမာ (၅ ကြိမ်ရေတွက်ခြင်း)


// ၁ မှ ၅ အထိ ရေတွက်ခြင်း
for (int i = 1; i <= 5; i++) { 
    System.out.println("Iteration: " + i);
}

Output ရလဒ်

Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
Iteration: 5

၃။ The `while` Loop

အကြိမ်ရေကို ကြိုတင်မသိဘဲ၊ **အခြေအနေတစ်ခု မှန်နေသမျှကာလပတ်လုံး** ဆက်လက်လည်ပတ်လိုသည့်အခါ while loop ကို အသုံးပြုသည်။ ၎င်းသည် **Loop မစတင်မီ** Condition ကို အရင် စစ်ဆေးသည်။

`while` Loop ဥပမာ


int count = 0;
// count သည် 3 ထက်ငယ်နေသမျှ ဆက်လည်ပတ်မည်
while (count < 3) { 
    System.out.println("Count: " + count);
    count++; // Loop ရပ်တန့်နိုင်ရန် တန်ဖိုးကို တိုးပေးရန် အရေးကြီးသည်
}

Output ရလဒ်

Count: 0
Count: 1
Count: 2

၄။ The `do-while` Loop

do-while loop သည် **Condition ကို နောက်ဆုံးမှ စစ်ဆေး** သောကြောင့်၊ Loop သည် **အနည်းဆုံး တစ်ကြိမ်** တော့ အမြဲတမ်း အလုပ်လုပ်မည်ဖြစ်သည်။

`do-while` Loop ဥပမာ


int x = 10;

do {
    // Condition (x < 5) မှားနေသော်လည်း ဤ Code သည် တစ်ကြိမ် အလုပ်လုပ်မည်။
    System.out.println("This runs once."); 
    x++;
} while (x < 5); // Condition သည် False ဖြစ်သည်

Output ရလဒ်

This runs once.
**Infinite Loop (အဆုံးမဲ့ Loop)** မဖြစ်စေရန် သတိထားပါ။ while နှင့် do-while တွင် Condition ကို False ဖြစ်သွားစေမည့် Variable (ဥပမာ: count++) ကို Code Block အတွင်း ထည့်သွင်းရန် မမေ့ပါနှင့်။
