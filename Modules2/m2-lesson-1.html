Lesson 2.1: If, Else If, Else Statements (ဆုံးဖြတ်ချက်များ)

ကြာမြင့်ချိန်: 25 မိနစ် | အဆင့်: အခြေခံ

၁။ Control Flow ဆိုသည်မှာ ဘာလဲ?

Control Flow ဆိုသည်မှာ Program တစ်ခုအတွင်းရှိ Code များကို မည်သည့်အစီအစဉ်အတိုင်း လုပ်ဆောင်မည်ကို သတ်မှတ်ခြင်းဖြစ်သည်။ **if/else** statements များသည် Program အား အခြေအနေ (Condition) များပေါ်မူတည်၍ လမ်းကြောင်းပြောင်းလဲ ဆုံးဖြတ်နိုင်စွမ်းကို ပေးသည်။

၂။ The `if` Statement

သတ်မှတ်ထားသော အခြေအနေ (Condition) သည် မှန်ကန်သည် (true) ဆိုပါက၊ Code Block တစ်ခုကို လုပ်ဆောင်ရန်အတွက် အသုံးပြုသည်။

`if` Statement ဥပမာ


int score = 75;

if (score > 70) {
    System.out.println("ဂုဏ်ယူပါတယ်၊ သင် အောင်မြင်ပါသည်!");
}

Output ရလဒ်

ဂုဏ်ယူပါတယ်၊ သင် အောင်မြင်ပါသည်!

၃။ The `if-else` Statement

အခြေအနေ (Condition) သည် မှန်ကန်လျှင် (`true` ဖြစ်လျှင်) **`if`** Code Block ကို လုပ်ဆောင်မည်။ မှားယွင်းလျှင် (`false` ဖြစ်လျှင်) **`else`** Code Block ကို လုပ်ဆောင်မည်။

`if-else` Statement ဥပမာ


int age = 16;
String eligibility;

if (age >= 18) {
    eligibility = "မဲပေးခွင့် ရှိသည်";
} else {
    eligibility = "မဲပေးခွင့် မရှိသေးပါ";
}

System.out.println("ရလဒ်: " + eligibility);

Output ရလဒ်

ရလဒ်: မဲပေးခွင့် မရှိသေးပါ

၄။ The `if-else if-else` Statement

အခြေအနေများစွာကို စစ်ဆေးလိုသည့်အခါ အသုံးပြုသည်။ ပထမဆုံး `true` ဖြစ်သော Condition ၏ Code Block ကိုသာ လုပ်ဆောင်ပြီး ကျန်တာများကို ကျော်သွားသည်။

`if-else if-else` Statement ဥပမာ (Grade စစ်ခြင်း)


int marks = 85;
String grade;

if (marks >= 90) {
    grade = "A";
} else if (marks >= 80) { // 80 မှ 89 အတွင်းသာ စစ်ဆေးသည်
    grade = "B";
} else if (marks >= 70) {
    grade = "C";
} else {
    grade = "F";
}

System.out.println("သင်ရရှိသော Grade: " + grade);

Output ရလဒ်

သင်ရရှိသော Grade: B
Condition များကို == (တူညီသည်), > (ပိုကြီးသည်), < (ပိုသေးသည်), && (AND), || (OR) စသည့် **Logical Operators** များဖြင့် စစ်ဆေးရမည်။
