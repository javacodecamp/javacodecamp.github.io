Lesson 2.5: Break နှင့် Continue Keywords (Loop ကို ထိန်းချုပ်ခြင်း)

ကြာမြင့်ချိန်: 10 မိနစ် | အဆင့်: အခြေခံ

၁။ Break Keyword ဆိုသည်မှာ ဘာလဲ?

break keyword ကို Loop တစ်ခုမှ **ချက်ချင်းထွက်ခွာရန်** အသုံးပြုသည်။ Condition မပြီးဆုံးသေးသော်လည်း break ကို တွေ့ရှိပါက Loop သည် ထိုနေရာတွင်ပင် ရပ်တန့်သွားမည်ဖြစ်သည်။

Break ဥပမာ (ဂဏန်း ၃ ရောက်လျှင် ရပ်ရန်)


for (int i = 1; i <= 10; i++) {
    if (i == 3) {
        break; // i သည် 3 နှင့်တူလျှင် Loop မှ ထွက်မည်
    }
    System.out.println("နံပါတ်: " + i);
}

Output ရလဒ်

နံပါတ်: 1
နံပါတ်: 2

၂။ Continue Keyword ဆိုသည်မှာ ဘာလဲ?

continue keyword သည် Loop ကို ရပ်တန့်ခြင်းမပြုဘဲ **လက်ရှိ အကြိမ် (Current Iteration) ကိုသာ ကျော်သွားရန်** အသုံးပြုသည်။ ၎င်းကို တွေ့ပါက အောက်ရှိ Code များကို မလုပ်ဆောင်ဘဲ နောက်တစ်ကြိမ် (Next Cycle) သို့ ချက်ချင်း ကူးပြောင်းသွားမည်ဖြစ်သည်။

Continue ဥပမာ (ဂဏန်း ၃ ကို ကျော်ရန်)


for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // i သည် 3 ဖြစ်ပါက အောက်က print ကို မလုပ်ဘဲ i=4 သို့ ကျော်မည်
    }
    System.out.println("နံပါတ်: " + i);
}

Output ရလဒ်

နံပါတ်: 1
နံပါတ်: 2
နံပါတ်: 4
နံပါတ်: 5

၃။ နှိုင်းယှဉ်ချက် အကျဉ်းချုပ်

Keywordအကျိုးသက်ရောက်မှု
breakLoop တစ်ခုလုံးကို ရပ်ပစ်ပြီး အပြင်သို့ ထွက်သည်။
continueလက်ရှိအဆင့်ကို ကျော်ပြီး နောက်တစ်ကြိမ် လည်ပတ်ရန် ပြန်သွားသည်။
break ကို switch statement များတွင်လည်း Case တစ်ခုချင်းစီကို ခွဲခြားရန် အသုံးပြုကြောင်း မှတ်သားထားပါ။
