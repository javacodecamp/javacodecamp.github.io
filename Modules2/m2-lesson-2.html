Lesson 2.2: Switch Statement (ရွေးချယ်မှုများ)

ကြာမြင့်ချိန်: 18 မိနစ် | အဆင့်: အခြေခံ

၁။ Switch Statement ဆိုသည်မှာ ဘာလဲ?

Variable တစ်ခု၏ တန်ဖိုးပေါ်မူတည်ပြီး ဖြစ်နိုင်ချေရှိသော ရွေးချယ်စရာ (Cases) အများအပြားထဲမှ တစ်ခုကို ရွေးချယ်လုပ်ဆောင်စေချင်သည့်အခါ switch ကို အသုံးပြုသည်။ ၎င်းသည် if-else if များကို အများကြီးသုံးခြင်းထက် ပိုမိုဖတ်ရလွယ်ကူစေသည်။

၂။ Switch ၏ ဖွဲ့စည်းပုံနှင့် Break Keyword

switch ထဲတွင် break keyword သည် အလွန်အရေးကြီးသည်။ Case တစ်ခုနှင့် ကိုက်ညီ၍ အလုပ်လုပ်ပြီးနောက် break မပါလျှင် အောက်က Case များကိုပါ ဆက်လက်လုပ်ဆောင်သွားမည်ဖြစ်သည်။ (ယင်းကို Fall-through ဟုခေါ်သည်)

Switch ဥပမာ (ရက်သတ္တပတ် နေ့ရက်များ)


int day = 3;
String dayName;

switch (day) {
    case 1: dayName = "Monday"; break;
    case 2: dayName = "Tuesday"; break;
    case 3: dayName = "Wednesday"; break;
    case 4: dayName = "Thursday"; break;
    case 5: dayName = "Friday"; break;
    default: dayName = "Weekend"; break;
}

System.out.println("ဒီနေ့က: " + dayName);

Output ရလဒ်

ဒီနေ့က: Wednesday

၃။ Switch with String (စာသားများဖြင့် အသုံးပြုခြင်း)

Java SE 7 မှစတင်၍ switch ထဲတွင် int, char များသာမက String များကိုပါ စစ်ဆေးနိုင်သည်။

String အသုံးပြုပုံ ဥပမာ


String level = "Expert";

switch (level) {
    case "Beginner":
        System.out.println("အခြေခံမှ စတင်လေ့လာပါ");
        break;
    case "Expert":
        System.out.println("ပရောဂျက်များ စတင်ရေးသားပါ");
        break;
    default:
        System.out.println("အဆင့် မသိပါ");
        break;
}

Output ရလဒ်

ပရောဂျက်များ စတင်ရေးသားပါ

၄။ Default Case

အကယ်၍ ပေးထားသော Case တန်ဖိုးများထဲမှ တစ်ခုနှင့်မျှ မကိုက်ညီပါက default အောက်ရှိ Code Block ကို လုပ်ဆောင်မည်ဖြစ်သည်။ ၎င်းသည် if-else ရှိ else နှင့် အလားသဏ္ဌာန်တူသည်။

switch ကို byte, short, char, int, String နှင့် Enums အမျိုးအစားများတွင်သာ သုံးနိုင်သည်။ double သို့မဟုတ် float တို့တွင် သုံး၍မရပါ။
