Lesson 2.4: Nested Loops (အဆင့်ဆင့် ပတ်လည်ခြင်း)

ကြာမြင့်ချိန်: 15 မိနစ် | အဆင့်: အခြေခံ

၁။ Nested Loop ဆိုသည်မှာ ဘာလဲ?

Nested Loop ဆိုသည်မှာ **Loop တစ်ခု၏ အတွင်း၌ နောက်ထပ် Loop တစ်ခု ထပ်မံ ထည့်သွင်း အသုံးပြုခြင်း** ဖြစ်သည်။ အပြင်ဘက် Loop (Outer Loop) တစ်ကြိမ် လည်ပတ်တိုင်း အတွင်းဘက် Loop (Inner Loop) သည် ၎င်း၏ သတ်မှတ်ထားသော အကြိမ်ရေ ပြည့်သည်အထိ အလုပ်လုပ်သည်။

၂။ အလုပ်လုပ်ပုံ အဆင့်ဆင့်

အပြင်ဘက် Loop တစ်ခါ လည်ပတ်လျှင် အတွင်းဘက် Loop က လိုအပ်သလောက် အကြိမ်ကြိမ် လည်ပတ်သည်။ အတွင်း Loop ပြီးမှသာ အပြင် Loop က ဒုတိယအကြိမ်သို့ ကူးပြောင်းသည်။

ဥပမာ ၁- ရိုးရှင်းသော Coordinate ပုံစံ


// Outer loop (စာကြောင်း)
for (int i = 1; i <= 2; i++) {
    
    // Inner loop (ကော်လံ)
    for (int j = 1; j <= 3; j++) {
        System.out.print("R" + i + "C" + j + " ");
    }
    
    System.out.println(); // တစ်ကြောင်းပြီးတိုင်း စာကြောင်းအသစ်ဆင်းရန်
}

Output ရလဒ်

R1C1 R1C2 R1C3 
R2C1 R2C2 R2C3

၃။ လက်တွေ့ အသုံးချပုံ (Pattern Drawing)

Nested Loop များကို အသုံးပြုပြီး ကြယ်ပွင့်ပုံစံများ (Stars patterns) ကို အလွယ်တကူ ရေးသားနိုင်သည်။

ဥပမာ ၂- ကြယ်ပွင့် တြိဂံပုံစံ ရေးဆွဲခြင်း


int rows = 4;

for (int i = 1; i <= rows; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print("* ");
    }
    System.out.println();
}

Output ရလဒ်

* * * * * * * * * *
**သတိထားရန်:** Nested Loops များသည် များပြားလာပါက (ဥပမာ- ၃ ထပ်၊ ၄ ထပ်) Program ၏ အလုပ်လုပ်နှုန်း (Performance) ကို သိသိသာသာ နှေးကွေးစေနိုင်ပါသည်။ ထို့ကြောင့် လိုအပ်မှသာ အသုံးပြုသင့်သည်။
