Lesson 5.1: Class နှင့် Object (အခြေခံ အယူအဆ)

ကြာမြင့်ချိန်: 30 မိနစ် | အဆင့်: အခြေခံ

၁။ Class ဆိုသည်မှာ ဘာလဲ?

Class ဆိုသည်မှာ Object တစ်ခု ဖန်တီးရန်အတွက် အသုံးပြုသော **Blueprint (ပုံစံကြမ်း)** ဖြစ်သည်။ ၎င်းတွင် Object တစ်ခု၌ ရှိရမည့် အချက်အလက် (Fields/Attributes) နှင့် လုပ်ဆောင်ချက် (Methods/Behaviors) များကို သတ်မှတ်ပေးထားသည်။

၂။ Object ဆိုသည်မှာ ဘာလဲ?

Object ဆိုသည်မှာ Class မှတစ်ဆင့် လက်တွေ့ ဖန်တီးလိုက်သော **Instance (အစစ်အမှန် အရာဝတ္ထု)** ဖြစ်သည်။ Class တစ်ခုတည်းမှနေ၍ Object ပေါင်းများစွာကို ဖန်တီးနိုင်သည်။

Class နှင့် Object ဥပမာ (Car Class)


// ၁။ Class ကို ကြေညာခြင်း
class Car {
    String brand;
    int year;

    void drive() {
        System.out.println(brand + " ကားလေး မောင်းနေပါပြီ။");
    }
}

public class Main {
    public static void main(String[] args) {
        // ၂။ Object ဖန်တီးခြင်း (Instantiating)
        Car myCar = new Car(); 
        
        // ၃။ အချက်အလက်များ ထည့်သွင်းခြင်း
        myCar.brand = "Toyota";
        myCar.year = 2022;

        // ၄။ လုပ်ဆောင်ချက်ကို ခေါ်ယူခြင်း
        myCar.drive();
    }
}

Output ရလဒ်

Toyota ကားလေး မောင်းနေပါပြီ။

၃။ Class နှင့် Object ၏ ကွာခြားချက်

Class (Blueprint) Object (Instance)
အယူအဆ (Logical Entity) သာဖြစ်သည်။ လက်တွေ့ရှိသော (Physical Entity) ဖြစ်သည်။
Memory နေရာယူမှု မရှိပါ။ ဖန်တီးလိုက်သောအခါ Memory နေရာယူသည်။
တစ်ကြိမ်သာ ကြေညာရသည်။ လိုအပ်သလောက် အကြိမ်ကြိမ် ဖန်တီးနိုင်သည်။
**မှတ်သားရန်:** Java တွင် new keyword ကို တွေ့လျှင် Object အသစ်တစ်ခုကို Memory (Heap) ပေါ်တွင် နေရာချ (Allocate) လိုက်ပြီဟု မှတ်သားနိုင်ပါသည်။
