Lesson 5.7: Interface နှင့် Abstract Class နှိုင်းယှဉ်ချက်

ကြာမြင့်ချိန်: 30 မိနစ် | အဆင့်: အလယ်အလတ်

၁။ Interface ဆိုသည်မှာ ဘာလဲ?

Interface ဆိုသည်မှာ Class တစ်ခုက ဘာတွေလုပ်နိုင်ရမည်လဲဆိုသည့် **"စာချုပ် (Contract)"** တစ်ခုကဲ့သို့ ဖြစ်သည်။ ၎င်းတွင် အကောင်အထည်ဖော်ရမည့် လုပ်ဆောင်ချက် (Method Signatures) များသာ ပါဝင်ပြီး၊ ၎င်းကို အသုံးပြုမည့် Class (Implementing Class) က အလုပ်လုပ်ပုံအသေးစိတ်ကို ရေးသားရမည်ဖြစ်သည်။

Interface ဥပမာ


interface Animal {
    void makeSound(); // abstract method (body မပါပါ)
}

class Dog implements Animal {
    public void makeSound() {
        System.out.println("ဝုတ် ဝုတ်!");
    }
}

၂။ Interface နှင့် Abstract Class ၏ ကွာခြားချက်များ

အဓိက ကွာခြားချက်များကို အောက်ပါဇယားတွင် ကြည့်ရှုနိုင်ပါသည်-

အချက်အလက် Abstract Class Interface
**Inheritance** Class တစ်ခုတည်းသာ ဆက်ခံနိုင်သည် (Extends)။ Interface အများအပြားကို တစ်ပြိုင်တည်း သုံးနိုင်သည်။ (Implements)
**Methods** Abstract နှင့် ပုံမှန် methods နှစ်မျိုးလုံး ပါနိုင်သည်။ Abstract methods သာ ပါဝင်နိုင်သည်။ (Java 8+ တွင် default/static ရနိုင်သည်)
**Variables** သာမန် variables များ ပါဝင်နိုင်သည်။ Constants (public static final) သာ ပါဝင်နိုင်သည်။
**Constructor** ရှိနိုင်သည်။ လုံးဝ မရှိနိုင်ပါ။

၃။ Multiple Inheritance (အမွေဆက်ခံမှု အများအပြား)

Java တွင် Class တစ်ခုသည် မိဘ Class တစ်ခုထက်ပို၍ အမွေဆက်ခံခွင့်မရှိသော်လည်း၊ Interface များကိုမူ အကန့်အသတ်မရှိ (Multiple Interfaces) အသုံးပြုခွင့်ပေးထားသည်။

Multiple Interfaces ဥပမာ


interface Printable { void print(); }
interface Showable { void show(); }

class Document implements Printable, Showable {
    public void print() { System.out.println("Printing..."); }
    public void show() { System.out.println("Showing..."); }
}

public class Main {
    public static void main(String[] args) {
        Document doc = new Document();
        doc.print();
        doc.show();
    }
}
**မည်သည့်အချိန်တွင် မည်သည်ကို သုံးရမည်နည်း?**
  • အကယ်၍ Class အချင်းချင်း ဆက်စပ်မှုရှိပြီး (is-a relationship) Code အချို့ကို ဝေမျှသုံးစွဲချင်လျှင် **Abstract Class** ကို သုံးပါ။
  • အကယ်၍ မတူညီသော Class များကြားတွင် တူညီသော စွမ်းဆောင်ရည် (Role/Capability) တစ်ခုကို သတ်မှတ်ချင်လျှင် **Interface** ကို သုံးပါ။
