Lesson 5.6: Constructor နှင့် အရေးကြီးသော Keywords များ

ကြာမြင့်ချိန်: 40 မိနစ် | အဆင့်: အလယ်အလတ်

၁။ Constructor ဆိုသည်မှာ ဘာလဲ?

Constructor ဆိုသည်မှာ Object တစ်ခုကို ဖန်တီးလိုက်သည့်အချိန်တွင် အလိုအလျောက် အလုပ်လုပ်သော အထူး Method တစ်ခု ဖြစ်သည်။ ၎င်းကို Variable တန်ဖိုးများကို စတင်သတ်မှတ်ရန် (Initialization) အတွက် အသုံးပြုသည်။

  • Constructor နာမည်သည် **Class နာမည်နှင့် အတူတူ** ဖြစ်ရမည်။
  • ၎င်းတွင် Return Type (void တောင်မှ) မရှိရပါ။

Constructor နှင့် `this` Keyword ဥပမာ


class Person {
    String name;

    // Constructor
    Person(String name) {
        // this.name သည် Class ၏ variable ကို ရည်ညွှန်းသည်
        this.name = name; 
        System.out.println("Object ဆောက်ပြီးပါပြီ။");
    }
}

public class Main {
    public static void main(String[] args) {
        Person p1 = new Person("Aung Aung");
        System.out.println("နာမည်: " + p1.name);
    }
}

Output ရလဒ်

Object ဆောက်ပြီးပါပြီ။
နာမည်: Aung Aung

၂။ `this` နှင့် `super` Keywords

`this` သည် လက်ရှိ Class အတွင်းရှိ Member များကို ရည်ညွှန်းပြီး၊ `super` သည် မိဘ Class (Superclass) ၏ Member များကို ရည်ညွှန်းသည်။

`super` အသုံးပြုပုံ ဥပမာ


class Animal {
    void eat() { System.out.println("Animal eats"); }
}

class Dog extends Animal {
    void eat() {
        super.eat(); // မိဘဆီက eat() ကို အရင်ခေါ်သည်
        System.out.println("Dog eats meat");
    }
}

၃။ `static` Keyword

static ဟု ကြေညာထားသော Member များသည် Object တစ်ခုချင်းစီနှင့် မဆိုင်ဘဲ **Class တစ်ခုလုံးနှင့် ဆိုင်သည်။** ၎င်းတို့ကို Object ဆောက်စရာမလိုဘဲ Class Name ဖြင့် တိုက်ရိုက်ခေါ်သုံးနိုင်သည်။

`static` ဥပမာ


class Counter {
    static int count = 0; // Class variable

    Counter() {
        count++;
    }
}

public class Main {
    public static void main(String[] args) {
        new Counter();
        new Counter();
        System.out.println("စုစုပေါင်း Object အရေအတွက်: " + Counter.count);
    }
}

Output ရလဒ်

စုစုပေါင်း Object အရေအတွက်: 2

၄။ `final` Keyword

final ကို ပြောင်းလဲ၍မရသော အရာများ သတ်မှတ်ရာတွင် သုံးသည်။

  • Variable: တန်ဖိုးကို ထပ်ပြင်၍မရပါ။ (Constant ဖြစ်သွားသည်)
  • Method: Subclass က Override လုပ်၍မရပါ။
  • Class: အမွေဆက်ခံ (Inherit) လုပ်၍မရပါ။

`final` ဥပမာ


final int MAX_SPEED = 120;
// MAX_SPEED = 150; // Error တက်မည် (တန်ဖိုးပြင်မရပါ)
**Pro Tip:** `static` နှင့် `final` ကို တွဲသုံးပြီး Java တွင် ကိန်းသေ (Constants) များကို သတ်မှတ်လေ့ရှိသည်။ ဥပမာ- `public static final double PI = 3.14159;`
