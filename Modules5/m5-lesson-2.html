Lesson 5.2: Inheritance (အမွေဆက်ခံခြင်း)

ကြာမြင့်ချိန်: 25 မိနစ် | အဆင့်: အလယ်အလတ်

၁။ Inheritance ဆိုသည်မှာ ဘာလဲ?

Inheritance ဆိုသည်မှာ Class တစ်ခု (Subclass) က အခြား Class တစ်ခု (Superclass) ၏ Properties (Attributes) နှင့် Methods များကို အမွေဆက်ခံရယူခြင်း ဖြစ်သည်။ ၎င်းကို **"is-a" relationship** ဟုလည်း ခေါ်သည်။ (ဥပမာ- Dog is an Animal)

[Image showing inheritance hierarchy with a Parent class (Animal) and Child classes (Dog, Cat) inheriting its traits]

၂။ Terminology (အခေါ်အဝေါ်များ)

  • **Superclass (Parent Class):** အမွေပေးသော Class (အခြေခံ Class)။
  • **Subclass (Child Class):** အမွေဆက်ခံသော Class (တိုးချဲ့ Class)။ ၎င်းသည် Superclass ၏ properties အပြင် ကိုယ်ပိုင် properties များကိုလည်း ထပ်တိုးနိုင်သည်။
  • **`extends` keyword:** အမွေဆက်ခံရန်အတွက် ဤ keyword ကို အသုံးပြုရသည်။

Inheritance ဥပမာ (Animal နှင့် Dog)


// ၁။ Superclass (မိဘ)
class Animal {
    void eat() {
        System.out.println("စားသောက်နေပါသည်...");
    }
}

// ၂။ Subclass (သားသမီး) - Animal ကို အမွေဆက်ခံသည်
class Dog extends Animal {
    void bark() {
        System.out.println("ဟောင်နေပါသည်... ဝုတ် ဝုတ်!");
    }
}

public class Main {
    public static void main(String[] args) {
        // Dog object ဆောက်သည်
        Dog myDog = new Dog();

        // မိဘဆီက အမွေရထားသော method ကို သုံးနိုင်သည်
        myDog.eat(); 

        // မိမိကိုယ်ပိုင် method ကို သုံးသည်
        myDog.bark();
    }
}

Output ရလဒ်

စားသောက်နေပါသည်...
ဟောင်နေပါသည်... ဝုတ် ဝုတ်!

၃။ အဘယ်ကြောင့် Inheritance ကို သုံးရသနည်း?

  1. **Code Reusability:** ရှိပြီးသား Code များကို ပြန်လည်အသုံးပြုနိုင်ခြင်း။
  2. **Method Overriding:** မိဘဆီက ရထားတဲ့ လုပ်ဆောင်ချက်ကို သားသမီးက စိတ်ကြိုက် ပြန်ပြင်ချင်တဲ့အခါ သုံးနိုင်ခြင်း (Polymorphism အတွက် အခြေခံ)။
  3. **Hierarchy:** Data များကို စနစ်တကျ အဆင့်ဆင့် စီစဉ်နိုင်ခြင်း။
**သတိထားရန်:** Java တွင် Class တစ်ခုသည် မိဘ Class **တစ်ခုတည်း** ဆီကသာ တိုက်ရိုက်အမွေဆက်ခံနိုင်သည်။ (Multiple Inheritance of classes is not supported to avoid complexity).
